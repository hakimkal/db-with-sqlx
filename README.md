### db-with-sqlx

> ** Quick Demo in Go ** with unit tests  of  sqlx and pgx for 
CRUD operations using clean code structure



> To Run tests and rest test coverage result. **Test coverage threshold: 70%**
  >> ./run_tests.sh


> Run code 
  >> 1. List Users ```go run cmd/main.go -taskName list```
> 
  >> 2. Get User By ID ```go run cmd/main.go -taskName view id 1```
